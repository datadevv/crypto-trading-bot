# Crypto Trading Bot - Golden Cross Strategy

Automated trading system developed in **Python** for the **QuantConnect** platform.

### Technical Specifications
* **Logic:** Simple Moving Average Crossover (10 & 30 days).
* **Risk Management:** Dynamic **5% Trailing Stop Loss** for profit protection.
* **Assets:** Multi-asset implementation for **BTC, ETH, and SOL**.

### Performance and Metrics
* **Probabilistic Sharpe Ratio (PSR):** 70%.
* **Focus:** High volatility resilience and automated trend following.
